#  Todo App - Daily Task Manager

A simple yet efficient daily task management application with browser-based storage. This project is built with pure HTML, CSS, and Vanilla JavaScript.

##  Features

- **Add Task**: With an optional date picker
- **Edit & Delete**: Each task can be edited or removed
- **Toggle Status**: Mark tasks as Done or Undo
- **Filter Tasks**: View All / Pending / Completed tasks
- **Delete All**: Clear the entire task list with one click
- **Auto-Save**: Automatically stores data in `localStorage` – no data loss on page refresh or closing the browser
- **Alert Notifications**: Success or error messages for every action

##  Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Responsive Table)
- JavaScript (ES6)
- Font: Arial (system default)
